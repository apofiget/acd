Description

Arduino serial port low level communication.
Process communicate with Arduino via serial port that name placed in app.config as {tty, "/path/to/dev/file"}.
Module write data to serial port and read data from port per line.

read/3

read(FD::pid(), Parent::pid(), LineEnd::string()) -> ok

Start serial port reader. Read data from serial port, that identified as low-level process pid - FD, and send data when line ends to parent process. Function should't be used manualy!
